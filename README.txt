Br-Br-Br-Br-Br-Beakout!!!!!!! is a rendition of the classic Atari game, "Breakout" created with the npgame library.

Controls on the keyboard are:
WASD
IJKL

'q' - quit level
'spacebar' - extra balls
'esc' - quit game
